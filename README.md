# CampusConnect 
**A Centralized Digital Campus & Student Lifecycle Management System**

## Description
In many educational institutions, student-faculty interactions are hindered by fragmented systems, manual attendance tracking, and the inefficient "come tomorrow" cycle. **CampusConnect** is an industry-level Enterprise Resource Planning (ERP) platform designed to digitize and streamline campus operations. 

By acting as a single source of truth, this platform eliminates data silos and provides transparent, role-based access for Students, Faculty, and Administrators.

### Key Features
* **Role-Based Access Control (RBAC):** Tailored dashboards and permissions for Admin, Faculty, and Students.
* **Hybrid Attendance Engine:** *One-Click Dashboard:* A rapid, manual override system where faculty only mark absentees.
  * *Secure Dynamic QR System:* An automated, anti-proxy attendance feature utilizing 15-second regenerating tokens and geolocation to ensure data integrity for large lecture halls.
* **Smart Appointment Booking:** A dedicated scheduling portal to streamline faculty office hours, eliminating unstructured drop-ins.
* **Real-Time Analytics:** Live campus engagement overviews and consolidated reporting for administrators.
